#### اضافه کردن کانال جدید

یا یک issue بسازید و لینک کانال و دسته‌بندی پیشنهادی رو بذارید تا اضافه کنم.

لیست بالا هر چند روز به صورت خودکار بازنویسی خواهد شد. در حال حاضر کانال شما باید دارای حداقل ۵ ویدئو و ۱۰ عضو باشد تا در لیست قرار گیرد. (این تعداد می‌تواند در آینده تغییر کند)
